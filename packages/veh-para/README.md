# @infernus/veh-para

[![npm](https://img.shields.io/npm/v/@infernus/veh-para)](https://www.npmx.dev/package/@infernus/veh-para) ![npm](https://img.shields.io/npm/dw/@infernus/veh-para) ![npm bundle size](https://img.shields.io/bundlephobia/minzip/@infernus/veh-para)

A wrapper of the popular [vehPara library](https://github.com/PutuSuhartawan/Scripting-Pawn-SAMP-Belajar-Dasar/blob/main/Set/VehPara.inc) for samp-node.

## Getting started

```sh
pnpm add @infernus/core @infernus/veh-para @infernus/colandreas
```

## Notice

- Mandatory use streamer, colandreas
